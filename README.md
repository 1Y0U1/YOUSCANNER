# YOUSCANNER

YOUSCANNER is ruby based port and services scanner for hacker or pentesters.

#Installation
sudo git clone https://github.com/1Y0U1/YOUSCANNER.git
cd YOUSCANNER
sudo chmod +x setup.sh
sudo bash setup.sh

#Mandatory Argument
-t  and -p   

#Optional Arguments

-h or --help -> For Help
-a  -> For all ports
-r  -> Port Number Range start-End
